    • Prosjekt i IN1050 - Introduksjon til design, bruk og interaksjon


    During this course, we were expected to produce high-fidelity prototypes with different approaches, using user-research, methods, and evaluations commonly applied in HCI (Human-Computer Interaction). In this project, I worked with the target group aged 50-60 years who use a home office for a large part of their workday.

    Research and analysis

    I conducted interviews with the target group about their daily experiences with working from home, focusing on work habits, workflow, and well-being. I transcribed the responses from these interviews and color-coded the data into different categories. From this data, I created an affinity diagram based on the observations made in each category

    From the affinity diagrams, I was able to identify some user needs that I could use to create prototypes that would meet those needs.

    Prototypes

    The sketch is based on the desktop app. It has four main features, represented by icons in the top left corner. These icons allow access to different pages: a question page where users can anonymously ask questions to colleagues, a chat page, a feed page where users can post pictures from their workday and interact with other colleagues, and a profile page. Below the icons on the right side, there is a list of recent contacts, which can be slid away.

    The app is designed to make it easier to stay in touch with colleagues throughout the day. It works by allowing users to add contacts and communicate with colleagues by navigating through different pages. There's a profile page where users can add nearby contacts, view a list of all contacts, and edit their profile. On the chat page, users can talk with colleagues. The question page allows users to ask colleagues for help with any issues. The feed page lets users post and view photos from their workday. Users interact with the app by clicking on the various icons at the top left of the screen, which take them to the different pages.
